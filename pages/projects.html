My projects

This page contains some lists of things I've worked on over the years in my spare time.

More information can be found about each entry at their respective sites (usually a hosted Git repository). The more interesting stuff might get their own articles in the future, which will be linked on this page as well as in the sidebar.

Electronics

  • DOOMbox digital, rp2350, c, assembly
    GitLab     A work-in-progress console dedicated to running a purpose-built DOOM clone. This is an evolution and hardware-based equivalent of my DOOM kernel, which is listed in another entry below.

Programming

Super low-level things

  • DOOM Kernel x86-assembly, nasm
    GitLab     A DOOM clone fully written in pure x86, Intel-syntax (NASM) assembly.
  • AWM c, unix, window-manager
    GitLab     A lightweight Linux X11 window manager written in C with libxcb.

Graphics and visualisations

  • The Thallium library c, 3dgraphics, vulkan, library
    GitLab     A low-level C library to abstract over multiple graphics/compute APIs, including Vulkan and OpenGL.
  • Minecraft with Vulkan c++, 3dgraphics, vulkan
    GitHub     An extremely work-in-progress implementation of Minecraft written in C++ rendered with Vulkan.

Utilities

  • Devinit rust, utility, cli
    GitHub     A command-line tool written in Rust to quickly initialise files and folders with preconfigured boilerplate templates. Note that I have started working on a rewrite of this with the goal of simplifying its interface for better usability: see its successor, Printing Press, on GitLab.
  • TexEdit c++, rust, latex, wxwidgets
    GitLab     An integrated editor and viewer (implemented in C++ with wxWidgets) and purpose-built compiler (CLI tool written in Rust) for TeX/LaTeX documents.
  • jSin javascript, browser-extension
    GitHub     A JavaScript injector extension, which I unfortunately had to abandon following Google burying Manifest V3 in their usual incompetence. It is available for installation from the Mozilla add-ons store for real browsers.

Other stuff

  • C Utils c, utility, library
    GitLab     A collection of very minimal/lightweight C libraries for data structures and I/O.
  • The Zeta maths library c, 3dgraphics, utility, library
    GitHub     A much older C library for 3D graphics-related maths operations.
